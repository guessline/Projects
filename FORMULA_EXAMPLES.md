# Примеры формул и расчётов

## 📐 Формулы листа "Ввод_данных"

### Базовые расчёты (строка 2 и далее)

```excel
# Ошибка (Error)
E2: =C2-D2
Пример: Факт=100, Прогноз=95 → Ошибка=5

# Абсолютная ошибка
F2: =ABS(E2)
Пример: Ошибка=-5 → |e|=5

# Процент абсолютной ошибки (защита от деления на ноль)
G2: =IF(C2>0;F2/C2;NA())
Пример: |e|=5, Факт=100 → %AbsErr=5%
Если Факт=0 → %AbsErr=#Н/Д
```

## 📊 Формулы листа "Метрики"

### 1. MAPE (Mean Absolute Percentage Error)

```excel
=IFERROR(AVERAGE(IF(ISNUMBER(Ввод_данных!G2:G1000);Ввод_данных!G2:G1000));NA())
```

**Что делает:**
1. Проверяет, является ли значение числом (ISNUMBER) — исключает #Н/Д
2. Берёт среднее только по числовым значениям
3. При отсутствии данных возвращает NA()

**Пример расчёта:**
```
Период 1: |e|=5,  y=100 → 5%
Период 2: |e|=3,  y=50  → 6%
Период 3: |e|=0,  y=0   → #Н/Д (исключается)
Период 4: |e|=10, y=200 → 5%

MAPE = (5% + 6% + 5%) / 3 = 5.33%
```

### 2. WAPE (Weighted Absolute Percentage Error)

```excel
=IF(SUM(Ввод_данных!C2:C1000)>0;SUM(Ввод_данных!F2:F1000)/SUM(Ввод_данных!C2:C1000);NA())
```

**Что делает:**
1. Суммирует все абсолютные ошибки
2. Делит на сумму всех фактических значений
3. Учитывает вес каждого периода по объёму

**Пример расчёта:**
```
Период 1: |e|=5,  y=100
Период 2: |e|=3,  y=50
Период 3: |e|=0,  y=0
Период 4: |e|=10, y=200

Сумма |e| = 5 + 3 + 0 + 10 = 18
Сумма y = 100 + 50 + 0 + 200 = 350

WAPE = 18 / 350 = 5.14%
```

**Преимущества WAPE:**
- Работает с нулевыми значениями
- Учитывает важность больших объёмов
- Не искажается при малых значениях

### 3. MAE (Mean Absolute Error)

```excel
=IFERROR(AVERAGE(IF(Ввод_данных!F2:F1000<>"";Ввод_данных!F2:F1000));NA())
```

**Что делает:**
1. Берёт среднее только по непустым значениям
2. Показывает среднюю ошибку в единицах (шт)

**Пример расчёта:**
```
Период 1: |e|=5
Период 2: |e|=3
Период 3: |e|=0
Период 4: |e|=10

MAE = (5 + 3 + 0 + 10) / 4 = 4.5 шт
```

### 4. RMSE (Root Mean Square Error)

```excel
=IFERROR(SQRT(AVERAGE((Ввод_данных!E2:E1000)^2));NA())
```

**Что делает:**
1. Возводит каждую ошибку в квадрат (усиливает выбросы)
2. Берёт среднее
3. Извлекает квадратный корень

**Пример расчёта:**
```
Период 1: e=5  → e²=25
Период 2: e=-3 → e²=9
Период 3: e=0  → e²=0
Период 4: e=10 → e²=100

Среднее e² = (25 + 9 + 0 + 100) / 4 = 33.5
RMSE = √33.5 = 5.79 шт
```

**Сравнение с MAE:**
```
MAE = 4.5 шт
RMSE = 5.79 шт
RMSE/MAE = 1.29

Если RMSE >> MAE (коэффициент >1.5) → есть выбросы
Если RMSE ≈ MAE → ошибки равномерны
```

### 5. Bias (Систематическое смещение)

```excel
=IFERROR(AVERAGE(IF(Ввод_данных!E2:E1000<>"";Ввод_данных!E2:E1000));NA())
```

**Что делает:**
1. Берёт среднее по ЗНАКОВЫМ ошибкам (не модуль!)
2. Показывает направление смещения

**Пример расчёта:**
```
Период 1: e=+5  (факт > прогноз)
Период 2: e=-3  (факт < прогноз)
Период 3: e=0   (точно)
Период 4: e=+10 (факт > прогноз)

Bias = (5 - 3 + 0 + 10) / 4 = +3 шт
```

**Интерпретация:**
- Bias > 0 → **Недопрогноз** (факт обычно больше прогноза)
  - Риск: дефициты, упущенные продажи
  - Действие: увеличить прогноз
  
- Bias < 0 → **Перепрогноз** (факт обычно меньше прогноза)
  - Риск: излишки, замороженный капитал
  - Действие: уменьшить прогноз
  
- Bias ≈ 0 → Смещения нет (но могут быть случайные ошибки)

## 🎯 Диагностические формулы

### Знак Bias

```excel
=IF(B8>0;"Недопрогноз (дефициты)";IF(B8<0;"Перепрогноз (излишки)";"Смещения нет"))
```

### Выявление выбросов

```excel
=IF(B7>B6;"Есть выбросы/крупные промахи";"Ошибки равномерны")
```

Где:
- B7 = RMSE
- B6 = MAE

### Рекомендация по WAPE

```excel
=IF(B5>0.15;"⚠ Проверьте сезонность, акции и качество данных";"✓ WAPE в норме")
```

## 🔍 Условное форматирование

### 1. Подсветка нулевых значений

```excel
Формула: =$C2=0
Формат: Жёлтая заливка
Применяется к: C2:C1000
```

**Зачем:** Нулевые факты делают MAPE неопределённым (деление на ноль).

### 2. Подсветка выбросов (топ-10%)

```excel
Формула: =$F2>=PERCENTILE($F$2:$F$1000;0.9)
Формат: Оранжевая заливка
Применяется к: F2:F1000
```

**Зачем:** Визуально выделяет крупные ошибки для анализа.

### 3. Цветовая шкала WAPE

```excel
WAPE < 10%   → Зелёный  (хорошо)
WAPE 10-15%  → Янтарный (приемлемо)
WAPE > 15%   → Красный  (требует внимания)
```

### 4. Индикация Bias

```excel
Bias > 0 → Синий   (недопрогноз)
Bias < 0 → Фиолетовый (перепрогноз)
```

## 📝 Примеры реальных кейсов

### Кейс 1: Стабильный товар

**Данные:**
```
Средние продажи: 100 шт/период
Стандартное отклонение: 10 шт
```

**Результаты:**
```
MAPE: 8%   → Точность приемлемая
WAPE: 7%   → Хорошо для отчётности
MAE: 8 шт  → В пределах σ
RMSE: 9 шт → Близко к MAE, выбросов нет
Bias: -1   → Минимальный перепрогноз
```

**Вывод:** Прогноз качественный, корректировки не требуются.

### Кейс 2: Товар с акциями

**Данные:**
```
Обычные периоды: 50 шт
Акционные периоды: 200 шт (не учтены в прогнозе)
```

**Результаты:**
```
MAPE: 35%  → Высокая ошибка
WAPE: 28%  → Критично
MAE: 40 шт → Большая средняя ошибка
RMSE: 85 шт → RMSE >> MAE (выбросы!)
Bias: +30  → Сильный недопрогноз
```

**Вывод:** 
1. Необходимо учитывать акционные периоды
2. RMSE показывает наличие выбросов
3. Положительный Bias → регулярно не хватает товара

**Действия:**
- Добавить флаг "акция" в модель
- Увеличить прогноз для акционных периодов
- Разделить анализ: обычные/акционные

### Кейс 3: Сезонный товар

**Данные:**
```
Зима: 200 шт
Лето: 50 шт
Прогноз не учитывает сезонность
```

**Результаты:**
```
MAPE: 45%   → Неприемлемо
WAPE: 40%   → Критично
MAE: 60 шт  → Большой разброс
RMSE: 75 шт → Есть выбросы
Bias: 0     → Компенсация + и - ошибок!
```

**Важно:** Bias=0 НЕ означает хороший прогноз! Зимой недопрогноз, летом перепрогноз.

**Вывод:**
- Bias маскирует проблему
- Нужна сезонная декомпозиция
- Анализировать отдельно по сезонам

## 📚 Математические основы

### MAPE - формула

$$MAPE = \frac{1}{n} \sum_{i=1}^{n} \frac{|y_i - f_i|}{y_i} \times 100\%$$

Где:
- $y_i$ — фактическое значение
- $f_i$ — прогнозное значение
- $n$ — количество периодов

### WAPE - формула

$$WAPE = \frac{\sum_{i=1}^{n} |y_i - f_i|}{\sum_{i=1}^{n} y_i} \times 100\%$$

### RMSE - формула

$$RMSE = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (y_i - f_i)^2}$$

### Bias - формула

$$Bias = \frac{1}{n} \sum_{i=1}^{n} (y_i - f_i)$$
