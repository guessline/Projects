## 🚀 **Готовая к использованию система:**

### 1️⃣ **Запуск Python моста:**
```bash
# Простой запуск
./run_production.sh

# Или напрямую
python3 mt5_bridge_production.py

# В фоне с логированием
nohup ./run_production.sh > ~/mt5_bridge.log 2>&1 &
```

### 2️⃣ **Установка в MT5:**
1. Скопируйте `mt5_expert_advisor.mq5` в:
   ```
   C:\Users\AdmVps\AppData\Roaming\MetaQuotes\Terminal\<TERMINAL_ID>\MQL5\Experts\
   ```
2. Откройте в MetaEditor
3. Компилируйте (F7)
4. Запустите на графике

### 3️⃣ **Настройка путей в EA:**
```mql5
// В настройках EA укажите:
InpFeaturesFile = "features_bt.csv"
InpPredictionsFile = "prediction_bt.txt"
```

## 📊 **Что получаете:**

### 🔄 **Процесс работы:**
1. **MT5 EA** записывает данные → `features_bt.csv`
2. **Python мост** читает данные → генерирует сигналы
3. **Python мост** записывает → `prediction_bt.txt` + `prediction_bt.json`
4. **MT5 EA** читает сигналы → выполняет сделки

### 📈 **Выходные данные:**

**Простой формат для MT5:**
```
BUY;2024.12.30 14:35
```

**Расширенный JSON для анализа:**
```json
{
  "signal": "BUY",
  "timestamp": "2024.12.30 14:35",
  "confidence": 0.85,
  "metadata": {
    "close": 1.0995,
    "ema": 1.0955,
    "atr": 0.0019,
    "spread_ratio": 0.0036
  }
}
```
