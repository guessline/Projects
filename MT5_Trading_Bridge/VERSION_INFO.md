## 🛠️ **Установка и Запуск**

### Базовая версия (без зависимостей):
```bash
python3 mt5_bridge.py
```

### Современная async версия:
```bash
python3 mt5_bridge_v2.py
```

### ML версия (требует установки):
```bash
pip install numpy pandas scikit-learn
python3 ml_analyzer.py
```

### Все зависимости одной командой:
```bash
pip install -r requirements.txt
```

## 🔗 **Интеграция с MT5**

1. Скопируйте `mt5_expert_advisor.mq5` в папку MT5: 
   ```
   MetaTrader 5/MQL5/Experts/
   ```

2. Компилируйте в MetaEditor (F7)

3. Запустите EA на графике

4. Запустите Python мост:
   ```bash
   ./start_bridge.sh
   ```
