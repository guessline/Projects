## 📊 Что такое метрики прогнозирования?

| Метрика | Что показывает | Когда использовать |
|---------|----------------|-------------------|
| **WAPE** | Взвешенная ошибка | ✅ Для отчётов и портфелей |
| **MAPE** | Средняя ошибка в % | ✅ Для сравнения моделей |
| **MAE** | Ошибка в штуках | ✅ Для простой оценки |
| **RMSE** | Учитывает выбросы | ✅ Для поиска аномалий |
| **Bias** | Направление ошибки | ✅ Всегда проверяйте! |

## ✅ Быстрая проверка качества прогноза

### Шаг 1: Посмотрите WAPE
- 🟢 **<10%** — Отлично, прогноз точный
- 🟡 **10-15%** — Приемлемо, можно улучшить
- 🔴 **>15%** — Плохо, нужны изменения

### Шаг 2: Проверьте Bias
- 🔵 **Положительный** — Недопрогноз (риск дефицитов)
- 🟣 **Отрицательный** — Перепрогноз (риск излишков)
- ⚪ **Около нуля** — Смещения нет

### Шаг 3: Сравните RMSE и MAE
- ✅ **RMSE ≈ MAE** — Ошибки равномерны
- ⚠️ **RMSE >> MAE** — Есть выбросы, нужен анализ
