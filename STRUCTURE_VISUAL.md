## 📊 Лист 2: "Ввод_данных"

```
┌────┬────────┬───────────┬──────────┬──────────┬──────────┬──────┬──────────┬─────────────┐
│ № │  SKU   │  Период   │ Факт_y   │Прогноз_f │ Ошибка_e │ |e|  │ %AbsErr  │ Примечание  │
│   │        │           │  (шт)    │  (шт)    │          │      │          │             │
├────┼────────┼───────────┼──────────┼──────────┼──────────┼──────┼──────────┼─────────────┤
│    │   A    │     B     │    C     │    D     │    E     │  F   │    G     │      H      │
├────┼────────┼───────────┼──────────┼──────────┼──────────┼──────┼──────────┼─────────────┤
│ 1  │  🔒    │    🔒     │   🟩     │   🟩     │   ⬜     │  ⬜  │   ⬜     │     🟩      │
├────┼────────┼───────────┼──────────┼──────────┼──────────┼──────┼──────────┼─────────────┤
│ 2  │TEST-001│01.01.2024 │   105    │   103    │    2     │  2   │  1.90%   │             │
├────┼────────┼───────────┼──────────┼──────────┼──────────┼──────┼──────────┼─────────────┤
│ 3  │TEST-001│01.02.2024 │   118    │   122    │   -4     │  4   │  3.39%   │             │
├────┼────────┼───────────┼──────────┼──────────┼──────────┼──────┼──────────┼─────────────┤
│... │  ...   │    ...    │   ...    │   ...    │   ...    │ ...  │   ...    │     ...     │
├────┼────────┼───────────┼──────────┼──────────┼──────────┼──────┼──────────┼─────────────┤
│13  │TEST-001│01.12.2024 │   93     │   95     │   -2     │  2   │  2.15%   │             │
└────┴────────┴───────────┴──────────┴──────────┴──────────┴──────┴──────────┴─────────────┘

Легенда:
  🟩 = Ввод (светло-зелёный, разблокировано)
  ⬜ = Формула (серый, защищено)
  🔒 = Заголовок (тёмно-серый)

Формулы:
  E2: =C2-D2                    ← Ошибка
  F2: =ABS(E2)                  ← Абсолютная ошибка
  G2: =IF(C2>0;F2/C2;NA())      ← % ошибки (защита от деления на 0)

Условное форматирование:
  🟨 Жёлтый: C=0 (y=0, MAPE не считается)
  🟧 Оранжевый: F в топ-10% (выбросы)

Диапазон данных: строки 2-1000 (формулы растянуты)
```

│          │          │          │ ошибка                 │ y=0             │
├──────────┼──────────┼──────────┼────────────────────────┼─────────────────┤
│  WAPE    │  7.32%   │    %     │ Взвешенная ошибка      │ Основная метрика│
│          │  🟩      │          │ по объёму              │ для портфелей   │
├──────────┼──────────┼──────────┼────────────────────────┼─────────────────┤

├──────────┼──────────┼──────────┼────────────────────────┼─────────────────┤
│  Bias    │  -0.5    │   шт     │ Знак = направление     │ - = перепрогноз │
│          │  🟣      │          │ смещения               │ + = недопрогноз │
└──────────┴──────────┴──────────┴────────────────────────┴─────────────────┘

┌───────────────────────────────────────────────────────────────────────────┐
│              ДИАГНОСТИКА И РЕКОМЕНДАЦИИ                                   │
├──────────────────────┬────────────────────────────────────────────────────┤
│  Знак Bias:          │  Перепрогноз (излишки)                             │
├──────────────────────┼────────────────────────────────────────────────────┤
│  RMSE vs MAE:        │  Ошибки равномерны                                 │
├──────────────────────┴────────────────────────────────────────────────────┤
│  РЕКОМЕНДАЦИИ:                                                            │
├──────────────────────┬────────────────────────────────────────────────────┤
│  Высокий WAPE:       │  ✓ WAPE в норме                                    │
├──────────────────────┼────────────────────────────────────────────────────┤
│  Разница RMSE-MAE:   │  ✓ Выбросов нет                                    │
├──────────────────────┼────────────────────────────────────────────────────┤

└──────────────────────┴────────────────────────────────────────────────────┘

Цветовая индикация:
  🟩 WAPE <10% (зелёный)
  🟨 WAPE 10-15% (янтарный)
  🟥 WAPE >15% (красный)
  
  🔵 Bias >0 (синий, недопрогноз)
  🟣 Bias <0 (фиолетовый, перепрогноз)
```

## 🚀 Сценарии использования

```
┌───────────────────────────────────────────────────────────┐
│              ТИПОВЫЕ СЦЕНАРИИ РАБОТЫ                      │
└───────────────────────────────────────────────────────────┘

СЦЕНАРИЙ 1: Оценка точности модели
─────────────────────────────────────────────────
  1. Загрузить данные → Ввод_данных (C, D)
  2. Проверить WAPE → Метрики (B5)
  3. Если >15% → искать выбросы (🟧 ячейки)
  4. Анализ Bias → корректировка модели


СЦЕНАРИЙ 2: Сравнение моделей
─────────────────────────────────────────────────
  1. Создать копию листа "Ввод_данных"
  2. Модель A → Лист 1, Модель B → Лист 2
  3. Сравнить MAPE/WAPE обеих моделей
  4. Выбрать модель с min(WAPE)


СЦЕНАРИЙ 3: Выявление систематических ошибок
─────────────────────────────────────────────────
  1. Проверить Bias → Метрики (B8)
  2. Если 🔵 (Bias>0) → увеличить прогноз
  3. Если 🟣 (Bias<0) → уменьшить прогноз
  4. Пересчитать метрики


СЦЕНАРИЙ 4: Поиск причин низкого качества
─────────────────────────────────────────────────
  1. Сравнить RMSE и MAE → Метрики (B7, B6)
  2. Если RMSE >> MAE → есть выбросы
