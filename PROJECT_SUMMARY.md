### Лист 2: "Ввод_данных"
**Колонки:**
| Колонка | Название | Тип | Описание |
|---------|----------|-----|----------|
| A | SKU | Ввод | Артикул товара |
| B | Период | Ввод | Дата или номер периода |
| C | Факт_y (шт) | Ввод | Фактические продажи |
| D | Прогноз_f (шт) | Ввод | Прогнозные продажи |
| E | Ошибка_e | Формула | =C-D |
| F | \|e\| | Формула | =ABS(E) |
| G | %AbsErr | Формула | =IF(C>0;F/C;NA()) |
| H | Примечание | Ввод | Доп. информация |

**Формулы (строки 2-1000):**
```excel
E2: =C2-D2                    # Ошибка
F2: =ABS(E2)                  # Абсолютная ошибка
G2: =IF(C2>0;F2/C2;NA())      # % ошибки с защитой от деления на 0
```

**Условное форматирование:**
1. 🟨 Жёлтая заливка при C=0 (предупреждение о нулевом факте)
2. 🟧 Оранжевая заливка для топ-10% ошибок (выбросы)

**Валидация:**
- C:D ≥ 0 (только положительные числа)
- Сообщения об ошибках при некорректном вводе

**Цветовое оформление:**
- Светло-зелёный (#E2F0D9): ячейки ввода (A, B, C, D, H)
- Серый (#F2F2F2): вычисляемые ячейки (E, F, G)
- Тёмно-серый (#404040): заголовки

### Лист 3: "Метрики"
**Основные метрики:**

| Метрика | Формула | Единицы | Интерпретация |
|---------|---------|---------|---------------|
| **MAPE** | `=IFERROR(AVERAGE(IF(ISNUMBER(G:G);G:G));NA())` | % | Средняя относительная ошибка |
| **WAPE** | `=IF(SUM(C:C)>0;SUM(F:F)/SUM(C:C);NA())` | % | Взвешенная ошибка по объёму |
| **MAE** | `=IFERROR(AVERAGE(IF(F:F<>"";F:F));NA())` | шт | Средняя абсолютная ошибка |
| **RMSE** | `=IFERROR(SQRT(AVERAGE((E:E)^2));NA())` | шт | Среднеквадратичная ошибка |
| **Bias** | `=IFERROR(AVERAGE(IF(E:E<>"";E:E));NA())` | шт | Систематическое смещение |

**Диагностические формулы:**
1. Знак Bias:
   ```excel
   =IF(B8>0;"Недопрогноз (дефициты)";IF(B8<0;"Перепрогноз (излишки)";"Смещения нет"))
   ```

2. Сравнение RMSE vs MAE:
   ```excel
   =IF(B7>B6;"Есть выбросы/крупные промахи";"Ошибки равномерны")
   ```

3. Рекомендации:
   - Высокий WAPE (>15%): "Проверьте сезонность, акции и качество данных"
   - Разница RMSE/MAE: "Сгладьте выбросы или разнесите всплески"
   - Значимый |Bias|: "Сместите модель/перепараметризуйте"

**Условное форматирование:**
- WAPE: 🟩 <10% | 🟨 10-15% | 🟥 >15%
- Bias: 🔵 >0 (недопрогноз) | 🟣 <0 (перепрогноз)

## 📊 Примеры использования

### Кейс 1: Оценка точности прогноза продаж
1. Загрузить фактические и прогнозные данные
2. Посмотреть WAPE на листе "Метрики"
3. Если >15% → анализировать выбросы (оранжевые ячейки)

### Кейс 2: Сравнение нескольких моделей
1. Создать копию листа для каждой модели
2. Сравнить MAPE/WAPE между моделями
3. Выбрать модель с минимальным WAPE

### Кейс 3: Выявление систематических ошибок
1. Проверить Bias на листе "Метрики"
2. Если положительный → увеличить прогноз
3. Если отрицательный → уменьшить прогноз

### Кейс 4: Поиск причин плохого прогноза
1. Сравнить RMSE и MAE
2. Если RMSE >> MAE → есть выбросы
3. Найти оранжевые ячейки → исследовать причины
