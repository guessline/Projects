# Forecast Metrics Template - Шаблон расчёта метрик прогнозирования

## 📋 Описание

Excel-шаблон для расчёта и интерпретации ключевых метрик точности прогнозирования: **MAPE**, **WAPE**, **MAE**, **RMSE**, **Bias**. Включает защиту формул, подсветку ошибок, автоматическую диагностику и графики.

## ✨ Возможности

- ✅ Автоматический расчёт 5 ключевых метрик точности прогноза
- ✅ Защита формул от случайного изменения
- ✅ Подсветка ошибок и выбросов
- ✅ Интерпретация результатов и рекомендации
- ✅ Графики для визуального анализа
- ✅ Совместимость с Excel ≥ 2016 и Google Sheets
- ✅ Поддержка мульти-SKU анализа

## 📊 Структура файла

### 1. **Инструкции** 
Полное руководство по использованию, описание метрик, ограничений и цветовая легенда.

### 2. **Ввод_данных**
Таблица для ввода фактических и прогнозных данных:

| Колонка | Описание | Тип |
|---------|----------|-----|
| **SKU** | Артикул товара | Ввод |
| **Период** | Дата или номер периода | Ввод |
| **Факт_y** | Фактические продажи (шт) | Ввод |
| **Прогноз_f** | Прогнозные продажи (шт) | Ввод |
| **Ошибка_e** | Ошибка = Факт - Прогноз | Формула |
| **\|e\|** | Абсолютная ошибка | Формула |
| **%AbsErr** | Процентная ошибка | Формула |
| **Примечание** | Доп. информация | Ввод |

### 3. **Метрики**
Автоматический расчёт метрик с интерпретацией:

- **MAPE** (Mean Absolute Percentage Error) - средняя относительная ошибка
- **WAPE** (Weighted Absolute Percentage Error) - взвешенная ошибка по объёму  
- **MAE** (Mean Absolute Error) - средняя абсолютная ошибка
- **RMSE** (Root Mean Square Error) - среднеквадратичная ошибка
- **Bias** - систематическое смещение

### 4. **Графики**
- Линейный график «Факт vs Прогноз»
- Столбчатая диаграмма «Абсолютная ошибка по периодам»

## 🚀 Как использовать

### Шаг 1: Ввод данных
1. Откройте лист **«Ввод_данных»**
2. Заполните колонки **C (Факт_y)** и **D (Прогноз_f)** своими данными
3. При необходимости укажите SKU в колонке A и период в колонке B

### Шаг 2: Проверка
- **Жёлтая подсветка** = нули в факте (MAPE не будет рассчитан)
- **Оранжевая подсветка** = выбросы (топ-10% ошибок)

### Шаг 3: Анализ
Перейдите на лист **«Метрики»** для просмотра:
- Рассчитанных значений метрик
- Автоматической диагностики
- Рекомендаций по улучшению

### Шаг 4: Визуализация
Просмотрите графики на листе **«Графики»**

## 📐 Примеры формул

### Ошибка прогноза
```excel
=C2-D2
```
где C2 - факт, D2 - прогноз

### Процентная абсолютная ошибка (с защитой от нулей)
```excel
=IF(C2>0;F2/C2;NA())
```

### MAPE (средняя процентная ошибка)
```excel
=IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA())
```

### WAPE (взвешенная ошибка)
```excel
=IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())
```

### MAE (средняя абсолютная ошибка)
```excel
=IFERROR(AVERAGEIF(rngAbsE;"<>0");NA())
```

### RMSE (среднеквадратичная ошибка)
```excel
=IFERROR(SQRT(SUMPRODUCT(rngE;rngE)/COUNTIF(rngE;"<>0"));NA())
```

### Bias (систематическое смещение)
```excel
=IFERROR(AVERAGEIF(rngE;"<>0");NA())
```

## 🎯 Интерпретация результатов

### WAPE (основная метрика)
- 🟢 **< 10%** - отличная точность
- 🟡 **10-15%** - приемлемая точность
- 🔴 **> 15%** - требуется улучшение

### Bias (направление смещения)
- 🔵 **Bias > 0** - недопрогноз (риск дефицитов)
- 🟣 **Bias < 0** - перепрогноз (риск излишков)
- ⚪ **Bias ≈ 0** - смещение отсутствует

### RMSE vs MAE
- Если **RMSE >> MAE** → есть выбросы/крупные промахи
- Если **RMSE ≈ MAE** → ошибки распределены равномерно

## ⚠️ Важные ограничения

1. **MAPE не рассчитывается** для периодов с нулевым фактом
2. **Если доля нулей > 30%** - используйте WAPE вместо MAPE
3. **RMSE чувствителен к выбросам** - всегда сравнивайте с MAE
4. **Минимум 5-10 периодов** для корректного расчёта
