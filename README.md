# Forecast Metrics Template

## Описание

Excel-шаблон для расчёта и анализа ключевых метрик качества прогнозирования. Поддерживает Excel ≥ 2016 и Google Sheets.

## Метрики

Шаблон автоматически рассчитывает следующие метрики:

- **MAPE** (Mean Absolute Percentage Error) — средняя относительная ошибка
- **WAPE** (Weighted Absolute Percentage Error) — взвешенная ошибка по объёму
- **MAE** (Mean Absolute Error) — средняя абсолютная ошибка
- **RMSE** (Root Mean Squared Error) — среднеквадратичная ошибка
- **Bias** — систематическое смещение прогноза

## Структура файла

### 1. Лист "Инструкции"
Подробное описание работы с шаблоном, объяснение метрик, ограничения и цветовая легенда.

### 2. Лист "Ввод_данных"

**Колонки:**
- `A` — SKU (артикул товара)
- `B` — Период (дата/номер)
- `C` — Факт_y (шт) — *вводимое поле*
- `D` — Прогноз_f (шт) — *вводимое поле*
- `E` — Ошибка_e — *вычисляемое*
- `F` — |e| — *вычисляемое*
- `G` — %AbsErr — *вычисляемое*
- `H` — Примечание — *вводимое поле*

**Формулы:**

```excel
E2: =C2-D2                    // Ошибка
F2: =ABS(E2)                  // Абсолютная ошибка
G2: =IF(C2>0;F2/C2;NA())      // % ошибка (с защитой от деления на 0)
```

**Именованные диапазоны:**
- `rngY` → `Ввод_данных!$C$2:$C$1000` (факт)
- `rngF` → `Ввод_данных!$D$2:$D$1000` (прогноз)
- `rngE` → `Ввод_данных!$E$2:$E$1000` (ошибка)
- `rngAbsE` → `Ввод_данных!$F$2:$F$1000` (абсолютная ошибка)
- `rngPctAbsE` → `Ввод_данных!$G$2:$G$1000` (% ошибка)

### 3. Лист "Метрики"

**Формулы расчёта метрик:**

**MAPE:**
```excel
=IFERROR(AVERAGE(IF(ISNUMBER(Ввод_данных!G2:G1000);Ввод_данных!G2:G1000));NA())
```
*Массивная формула. Исключает строки с y=0*

**WAPE:**
```excel
=IF(SUM(Ввод_данных!C2:C1000)>0;SUM(Ввод_данных!F2:F1000)/SUM(Ввод_данных!C2:C1000);NA())
```
*Взвешенная метрика по всему объёму*

**MAE:**
```excel
=IFERROR(AVERAGE(IF(Ввод_данных!F2:F1000<>"";Ввод_данных!F2:F1000));NA())
```
*Среднее по непустым значениям*

**RMSE:**
```excel
=IFERROR(SQRT(AVERAGE((Ввод_данных!E2:E1000)^2));NA())
```
*Корень из средней квадратичной ошибки*

**Bias:**
```excel
=IFERROR(AVERAGE(IF(Ввод_данных!E2:E1000<>"";Ввод_данных!E2:E1000));NA())
```
*Среднее смещение (знак важен!)*

**Диагностические формулы:**

Интерпретация знака Bias:
```excel
=IF(B7>0;"Недопрогноз (дефициты)";IF(B7<0;"Перепрогноз (излишки)";"Смещения нет"))
```

Сравнение RMSE vs MAE:
```excel
=IF(B6>B5;"Есть выбросы/крупные промахи";"Ошибки равномерны")
```

### 4. Лист "Графики"

Визуализация:
- Линейный график "Факт vs Прогноз"
- Столбчатая диаграмма "Абсолютная ошибка по периодам"
- Карта статусов с автоматической оценкой качества

## Условное форматирование

**На листе "Ввод_данных":**
- 🟡 **Жёлтая заливка** — нулевой факт (y=0), MAPE не рассчитывается
- 🟠 **Оранжевая заливка** — выброс (ошибка в топ-10%)

**На листе "Метрики":**
- 🟢 **Зелёный** — WAPE < 10% (отлично)
- 🟡 **Янтарный** — WAPE 10-15% (средне)
- 🔴 **Красный** — WAPE > 15% (требует улучшения)

## Интерпретация результатов

### Когда использовать какую метрику:

**MAPE:**
- ✅ Для стабильных рядов без нулей
- ❌ Не используйте, если доля нулей > 30%
- 📊 Норма: <10% — отлично, 10-20% — хорошо, >20% — плохо

**WAPE:**
- ✅ Для портфелей SKU
- ✅ Для отчётности руководству
- ✅ Устойчива к выбросам
- 📊 Учитывает объём продаж

**MAE:**
- ✅ Показывает типичную ошибку в штуках
- ✅ Легко интерпретировать
- 📊 Сравнивайте с RMSE для выявления выбросов

**RMSE:**
- ✅ Когда большие ошибки критичнее малых
- ✅ Для оценки рисков дефицитов/излишков
- 📊 Если RMSE >> MAE → есть серьёзные выбросы

**Bias:**
- ✅ ВСЕГДА проверяйте знак!
- ➕ Положительный → недопрогноз → дефициты
- ➖ Отрицательный → перепрогноз → излишки
- 📊 |Bias| ≈ 0 не означает хороший прогноз, ошибки могут компенсироваться

### Рекомендации по улучшению:

| Проблема | Действие |
|----------|----------|
| Высокий WAPE (>15%) | Проверьте сезонность, акции, качество данных |
| RMSE >> MAE | Сгладьте выбросы или разнесите всплески |
| \|Bias\| значителен | Сместите модель или перепараметризуйте |
| MAPE = NA | Много нулей — используйте WAPE |

## Защита и безопасность

- 🔒 **Листы "Метрики" и "Графики"** — полностью защищены
- 🔓 **Лист "Ввод_данных"** — доступны для редактирования только:
  - Колонки A, B, C, D, H (светло-зелёный фон)
  - Остальные колонки с формулами защищены

*Защита без пароля для удобства использования*

## Валидация данных

- Колонки C и D (Факт/Прогноз): только числа ≥ 0
- Автоматическое предупреждение при вводе отрицательных значений
- Пустые ячейки допускаются

## Тестовые данные

Шаблон содержит 12 периодов тестовых данных для демонстрации работы:
- 1 SKU (SKU_TEST_001)
- Целевые значения метрик: MAPE ~8-12%, WAPE ~7-10%
- Bias небольшой (близок к нулю)

## Быстрый старт

1. **Откройте файл** `Forecast_Metrics_Template.xlsx`
2. **Перейдите на лист "Ввод_данных"**
3. **Замените тестовые данные** на свои:
   - Колонка C: фактические продажи
   - Колонка D: прогнозные продажи
4. **Перейдите на лист "Метрики"** — результаты обновятся автоматически
5. **Изучите "Графики"** для визуального анализа

## Ограничения

⚠️ **MAPE и нулевые значения:**
- MAPE не может быть рассчитан при y=0 (деление на ноль)
- Такие строки автоматически исключаются
- Если доля нулей >30% — используйте WAPE

⚠️ **Выбросы:**
- RMSE сильно реагирует на выбросы
- Проверяйте оранжевые ячейки (топ-10% ошибок)

⚠️ **Интерпретация Bias:**
- Малый |Bias| ≠ хороший прогноз
- Ошибки могут компенсироваться
- Всегда смотрите MAPE/WAPE вместе с Bias
