# Forecast Metrics Template

Excel-шаблон для расчёта и анализа метрик точности прогнозирования.

## 📋 Описание

Шаблон предназначен для автоматического расчёта основных метрик качества прогнозов:
- **MAPE** (Mean Absolute Percentage Error) — средняя относительная ошибка
- **WAPE** (Weighted Absolute Percentage Error) — взвешенная ошибка по объёму
- **MAE** (Mean Absolute Error) — средняя абсолютная ошибка
- **RMSE** (Root Mean Square Error) — среднеквадратичная ошибка
- **Bias** — систематическое смещение прогноза

## 🚀 Быстрый старт

1. Откройте файл `Forecast_Metrics_Template.xlsx`
2. Перейдите на лист **"Ввод_данных"**
3. Заполните колонки (светло-зелёные ячейки):

4. Перейдите на лист **"Метрики"** для просмотра результатов
5. Изучите графики на листе **"Графики"**

## 📊 Структура файла

### 1. Лист "Инструкции"
Подробное руководство по использованию шаблона, описание метрик и рекомендации.

### 2. Лист "Ввод_данных"
Таблица для ввода фактических и прогнозных данных.

**Колонки:**
| Колонка | Описание | Тип |
|---------|----------|-----|
| A: SKU | Артикул или название товара | Ввод |
| B: Период | Дата или номер периода | Ввод |
| C: Факт_y (шт) | Фактические продажи | Ввод |
| D: Прогноз_f (шт) | Прогнозные продажи | Ввод |
| E: Ошибка_e | Разница: Факт - Прогноз | Расчёт |
| F: \|e\| | Абсолютная ошибка | Расчёт |
| G: %AbsErr | Процент абсолютной ошибки | Расчёт |
| H: Примечание | Дополнительная информация | Ввод |

**Формулы:**
```excel
E2: =C2-D2                          # Ошибка
F2: =ABS(E2)                        # Абсолютная ошибка
G2: =IF(C2>0;F2/C2;NA())           # % ошибки (NA при y=0)
```

**Условное форматирование:**
- 🟨 Жёлтая заливка — нулевой факт (y=0), MAPE не считается
- 🟧 Оранжевая заливка — выбросы (топ-10% ошибок)

### 3. Лист "Метрики"
Автоматический расчёт всех метрик с интерпретацией.

**Формулы метрик:**

```excel
# MAPE — средняя относительная ошибка
=IFERROR(AVERAGE(IF(ISNUMBER(Ввод_данных!G2:G1000);Ввод_данных!G2:G1000));NA())

# WAPE — взвешенная ошибка по объёму
=IF(SUM(Ввод_данных!C2:C1000)>0;SUM(Ввод_данных!F2:F1000)/SUM(Ввод_данных!C2:C1000);NA())

# MAE — средняя абсолютная ошибка
=IFERROR(AVERAGE(IF(Ввод_данных!F2:F1000<>"";Ввод_данных!F2:F1000));NA())

# RMSE — среднеквадратичная ошибка
=IFERROR(SQRT(AVERAGE((Ввод_данных!E2:E1000)^2));NA())

# Bias — систематическое смещение
=IFERROR(AVERAGE(IF(Ввод_данных!E2:E1000<>"";Ввод_данных!E2:E1000));NA())
```

**Диагностика:**
- Знак Bias: положительный = недопрогноз (дефициты), отрицательный = перепрогноз (излишки)
- RMSE vs MAE: если RMSE >> MAE → есть выбросы
- Автоматические рекомендации по улучшению прогноза

**Цветовая индикация WAPE:**
- 🟩 Зелёный: <10% (хорошо)
- 🟨 Янтарный: 10-15% (приемлемо)
- 🟥 Красный: >15% (требует внимания)

## 📐 Когда использовать какую метрику

### MAPE
✅ **Применяйте когда:**
- Нужна относительная оценка (в %)
- Данные стабильные, без нулей
- Сравниваете точность разных моделей

❌ **Не используйте когда:**
- Есть нулевые фактические значения
- Доля нулей >30%
- Данные с высокой волатильностью

### WAPE
✅ **Применяйте когда:**
- Анализируете портфель SKU
- Важна точность по объёму продаж
- Готовите отчётность для руководства
- Есть нулевые значения в данных

### MAE
✅ **Применяйте когда:**
- Нужна простая и понятная метрика
- Важна средняя ошибка в штуках
- Нет необходимости в процентах

### RMSE
✅ **Применяйте когда:**
- Важно выявить крупные промахи
- Нужно усилить влияние выбросов
- Сравниваете с MAE для диагностики

### Bias
✅ **Всегда проверяйте:**
- Знак показывает направление смещения
- Положительный = недопрогноз (риск дефицитов)
- Отрицательный = перепрогноз (риск излишков)

## 📝 Примеры интерпретации

### Пример 1: Хороший прогноз
```
MAPE: 8%
WAPE: 7%
MAE: 5 шт
RMSE: 6 шт
Bias: -0.5 шт
```
**Интерпретация:** Точный прогноз с минимальным перепрогнозом. Выбросов нет (RMSE≈MAE).

### Пример 2: Требует внимания
```
MAPE: 25%
WAPE: 18%
MAE: 15 шт
RMSE: 35 шт
Bias: +12 шт
```
**Интерпретация:** Низкая точность, есть выбросы (RMSE>>MAE), систематический недопрогноз. 
**Рекомендации:** Проверить сезонность, добавить факторы спроса, скорректировать модель вверх.
