# Forecast Metrics Template

## 📊 Описание

Excel-шаблон для расчёта и интерпретации ключевых метрик точности прогнозирования:
- **MAPE** (Mean Absolute Percentage Error) — средняя абсолютная процентная ошибка
- **WAPE** (Weighted Absolute Percentage Error) — взвешенная абсолютная процентная ошибка
- **MAE** (Mean Absolute Error) — средняя абсолютная ошибка
- **RMSE** (Root Mean Square Error) — среднеквадратичная ошибка
- **Bias** — систематическое смещение прогноза

## 🚀 Быстрый старт

1. Откройте файл `Forecast_Metrics_Template.xlsx`
2. Перейдите на лист **"Ввод_данных"**
3. Заполните колонки:
   - **C (Факт_y)** — фактические продажи (шт)
   - **D (Прогноз_f)** — прогнозные значения (шт)
4. Перейдите на лист **"Метрики"** для просмотра результатов
5. Изучите визуализацию на листе **"Графики"**

## 📐 Примеры формул

### Лист "Ввод_данных"

**Ошибка прогноза (E2):**
```excel
=C2-D2
```

**Абсолютная ошибка (F2):**
```excel
=ABS(E2)
```

**Процентная ошибка с защитой от нулей (G2):**
```excel
=IF(C2>0;F2/C2;NA())
```
*Примечание: При y=0 возвращает #Н/Д для корректного расчёта MAPE*

### Лист "Метрики"

**MAPE — средняя процентная ошибка:**
```excel
=IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA())
```
*Массивная формула. В Excel нажмите Ctrl+Shift+Enter*

**WAPE — взвешенная процентная ошибка:**
```excel
=IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())
```

**MAE — средняя абсолютная ошибка:**
```excel
=IFERROR(AVERAGEIF(rngAbsE;">0");NA())
```

**RMSE — среднеквадратичная ошибка:**
```excel
=IFERROR(SQRT(AVERAGE(rngE^2));NA())
```

**Bias — систематическое смещение:**
```excel
=IFERROR(AVERAGE(rngE);NA())
```

**Диагностика знака Bias:**
```excel
=IF(Bias_value>0;"Недопрогноз (дефициты)";IF(Bias_value<0;"Перепрогноз (излишки)";"Смещения нет"))
```

**Сравнение RMSE vs MAE:**
```excel
=IF(RMSE_value>MAE_value;"Есть выбросы/крупные промахи";"Ошибки равномерны")
```

## 📈 Интерпретация метрик

### MAPE (Mean Absolute Percentage Error)
- **< 5%** — отличный прогноз
- **5-10%** — хороший прогноз
- **10-20%** — приемлемый прогноз
- **> 20%** — требуется улучшение

⚠️ **Ограничение:** Не используйте при нулевых фактических значениях или если доля нулей > 30%

### WAPE (Weighted Absolute Percentage Error)
- **< 10%** — отличный прогноз
- **10-15%** — хороший прогноз
- **> 15%** — требуется улучшение

✅ **Преимущество:** Устойчив к выбросам, учитывает объём продаж

### MAE (Mean Absolute Error)
- Показывает среднюю ошибку в штуках
- Простая интерпретация, устойчива к выбросам
- Используйте для оценки абсолютных отклонений

### RMSE (Root Mean Square Error)
- Усиливает влияние больших промахов
- **Если RMSE >> MAE** → присутствуют выбросы
- **Если RMSE ≈ MAE** → ошибки равномерны

### Bias (Систематическое смещение)
- **Bias > 0** → Недопрогноз (риск дефицитов)
- **Bias < 0** → Перепрогноз (риск излишков)
- **Bias ≈ 0** → Смещение отсутствует

## 🔒 Защита и валидация

### Защищённые листы
- **"Метрики"** — только просмотр
- **"Графики"** — только просмотр
- **"Инструкции"** — не защищён

### Редактируемые поля на листе "Ввод_данных"
- Колонка A — SKU
- Колонка B — Период
- Колонка C — Факт_y (с валидацией ≥ 0)
- Колонка D — Прогноз_f (с валидацией ≥ 0)
- Колонка H — Примечание

### Автоматическая валидация
- Факт и Прогноз должны быть ≥ 0
- Пустые ячейки допускаются
- При нарушении — всплывает сообщение об ошибке

## 📋 Условное форматирование

1. **Жёлтая заливка** (C2:C1000) — предупреждение о нулевом факте (y=0)
   - Формула: `=C2=0`
   
2. **Оранжевая заливка** (F2:F1000) — выбросы (топ-10% ошибок)
   - Формула: `=F2>=PERCENTILE($F$2:$F$1000;0.9)`

3. **Цветовая шкала WAPE** (B5 на листе "Метрики"):
   - Зелёный: < 10%
   - Янтарный: 10-15%
   - Красный: > 15%

4. **Индикатор Bias** (B8 на листе "Метрики"):
   - Синий: Bias > 0 (недопрогноз)
   - Фиолетовый: Bias < 0 (перепрогноз)

## 🛠️ Технические требования

- **Excel:** версия 2016 и выше
- **Google Sheets:** совместим
- **Разделитель аргументов:** `;` (точка с запятой)
- **Десятичный разделитель:** `,` (запятая) — для Windows RU

### Для других региональных настроек
Если используете точку как десятичный разделитель:
1. Замените `;` на `,` в формулах
2. Замените `,` на `.` для десятичных чисел

## ❓ Частые вопросы

**В: Почему MAPE показывает #Н/Д?**
О: В данных присутствуют нулевые фактические значения. Используйте WAPE.

**В: Как интерпретировать высокий RMSE при низком MAE?**
О: Присутствуют единичные большие промахи (выбросы). Проверьте оранжевые ячейки на листе "Ввод_данных".

**В: Bias отрицательный — это плохо?**
О: Отрицательный Bias означает перепрогноз. Это может привести к излишкам. Нужно скорректировать модель вниз.

**В: Можно ли добавить свои метрики?**
О: Да, на листе "Метрики" можно добавить дополнительные строки с формулами. Например, MASE или SMAPE.
