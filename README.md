# Forecast Metrics Template

**Шаблон для расчёта и интерпретации метрик точности прогнозирования**

## Описание

Excel-шаблон для анализа качества прогнозов с расчётом ключевых метрик:
- **MAPE** (Mean Absolute Percentage Error) — средняя абсолютная процентная ошибка
- **WAPE** (Weighted Absolute Percentage Error) — взвешенная абсолютная процентная ошибка  
- **MAE** (Mean Absolute Error) — средняя абсолютная ошибка
- **RMSE** (Root Mean Square Error) — среднеквадратичная ошибка
- **Bias** — смещение прогноза

## Совместимость

- Excel ≥ 2016
- Google Sheets
- Разделитель аргументов: `;` (точка с запятой)
- Десятичный разделитель: `,` (запятая)

## Структура файла

Шаблон содержит 4 листа:

### 1. Инструкции
Подробное руководство по работе с шаблоном, описание метрик, ограничения и рекомендации.

### 2. Ввод_данных
Таблица для ввода фактических и прогнозных значений:

| Колонка | Название | Описание | Тип |
|---------|----------|----------|-----|
| A | SKU | Артикул товара | Ввод |
| B | Период | Дата или номер периода | Ввод |
| C | Факт_y (шт) | Фактические продажи | Ввод |
| D | Прогноз_f (шт) | Прогнозные значения | Ввод |
| E | Ошибка_e | Ошибка = Факт - Прогноз | Формула |
| F | \|e\| | Абсолютная ошибка | Формула |
| G | %AbsErr | Процентная ошибка для MAPE | Формула |
| H | Примечание | Опциональные заметки | Ввод |

#### Основные формулы:

```excel
; Ошибка (E2)
=C2-D2

; Абсолютная ошибка (F2)
=ABS(E2)

; Процентная ошибка (G2) - защита от деления на 0
=IF(C2>0;F2/C2;NA())
```

#### Условное форматирование:
- 🟡 **Жёлтая заливка** — факт = 0 (MAPE не рассчитывается)
- 🟠 **Оранжевая заливка** — выбросы (ошибка в топ-10%)

### 3. Метрики
Расчёт метрик и диагностика качества прогноза.

#### Формулы метрик:

```excel
; MAPE - средняя процентная ошибка
=IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA())

; WAPE - взвешенная процентная ошибка
=IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())

; MAE - средняя абсолютная ошибка
=IFERROR(AVERAGE(IF(rngAbsE<>"";rngAbsE));NA())

; RMSE - среднеквадратичная ошибка
=IFERROR(SQRT(AVERAGE((rngE)^2));NA())

; Bias - смещение прогноза
=IFERROR(AVERAGE(IF(rngE<>"";rngE));NA())
```

#### Диагностические формулы:

```excel
; Интерпретация Bias
=IF(Bias>0;"Недопрогноз (дефициты)";IF(Bias<0;"Перепрогноз (излишки)";"Смещения нет"))

; Сравнение RMSE и MAE
=IF(RMSE>MAE;"Есть выбросы/крупные промахи";"Ошибки равномерны")

; Рекомендация при высоком WAPE
=IF(WAPE>0,15;"⚠ Проверьте сезонность/акции/качество данных";"-")

; Рекомендация при выбросах
=IF(RMSE>MAE*1,2;"⚠ Сгладьте выбросы или разнесите всплески";"-")

; Рекомендация при смещении
=IF(ABS(Bias)>MAE*0,3;"⚠ Сместите модель/перепараметризуйте";"-")
```

#### Условное форматирование метрик:
- 🟢 **WAPE < 10%** — отличный результат
- 🟠 **WAPE 10-15%** — приемлемо
- 🔴 **WAPE > 15%** — требует улучшения
- 🔵 **Bias > 0** — недопрогноз
- 🟣 **Bias < 0** — перепрогноз

### 4. Графики
Визуализация данных:
- **График 1**: Линейный график "Факт vs Прогноз"
- **График 2**: Столбчатый график "Абсолютная ошибка по периодам"
- **Карта статусов**: Сводная таблица со статусами метрик

## Как использовать

### Шаг 1: Ввод данных
1. Откройте лист **"Ввод_данных"**
2. Заполните колонки:
   - **A (SKU)** — артикул товара
   - **B (Период)** — дата или номер периода
   - **C (Факт_y)** — фактические продажи в штуках
   - **D (Прогноз_f)** — прогнозные значения в штуках
   - **H (Примечание)** — опционально

### Шаг 2: Проверка ошибок
- Обратите внимание на **жёлтые ячейки** (факт = 0, MAPE недоступен)
- Найдите **оранжевые ячейки** (выбросы в топ-10%)

### Шаг 3: Анализ метрик
- Перейдите на лист **"Метрики"**
- Изучите рассчитанные значения:
  - **MAPE** — для оценки относительной точности
  - **WAPE** — для портфельной оценки
  - **MAE** — для абсолютной величины ошибки
  - **RMSE** — для выявления выбросов
  - **Bias** — для определения направления смещения
- Прочитайте рекомендации в разделе "ДИАГНОСТИКА И РЕКОМЕНДАЦИИ"

### Шаг 4: Визуализация
- Откройте лист **"Графики"**
- Изучите визуализации и карту статусов

## Интерпретация метрик

### MAPE (Mean Absolute Percentage Error)
- **< 10%** — отличный прогноз
- **10-20%** — хороший прогноз
- **> 20%** — требует улучшения
- ⚠️ **Ограничение**: не рассчитывается при y=0

### WAPE (Weighted Absolute Percentage Error)
- **< 10%** — отлично (зелёный)
- **10-15%** — приемлемо (янтарный)
- **> 15%** — требует улучшения (красный)
- ✅ **Преимущество**: учитывает объём продаж

### MAE (Mean Absolute Error)
- Показывает среднюю ошибку в натуральных единицах (штуках)
- Используется для операционного планирования

### RMSE (Root Mean Square Error)
- Усиливает влияние больших ошибок
- Сравните с MAE:
  - **RMSE ≈ MAE** → ошибки равномерны
  - **RMSE >> MAE** → есть выбросы

### Bias (Смещение)
- **Bias > 0** → недопрогноз (риск дефицитов)
- **Bias < 0** → перепрогноз (риск излишков)
- **Bias ≈ 0** → смещения нет

## Важные замечания

### ⚠️ MAPE при нулевых значениях
Если доля нулей в факте **> 30%**, не используйте MAPE. Вместо него применяйте WAPE или MAE.

### ⚠️ Влияние выбросов на RMSE
RMSE более чувствителен к выбросам, чем MAE. Если RMSE значительно больше MAE — в данных есть крупные промахи.

### ⚠️ Смысл Bias
Низкий Bias не означает хороший прогноз — ошибки могут компенсировать друг друга. Всегда анализируйте Bias вместе с WAPE/MAE/RMSE.
