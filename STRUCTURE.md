### 2. 📝 ВВОД_ДАННЫХ
**Назначение:** Таблица для ввода фактических и прогнозных данных

**Структура колонок:**

| Колонка | Название | Тип | Описание | Цвет |
|---------|----------|-----|----------|------|
| A | SKU | Ввод | Артикул товара | 🟢 Зелёный |
| B | Период | Ввод | Дата или номер периода | 🟢 Зелёный |
| C | Факт_y (шт) | Ввод | Фактические продажи | 🟢 Зелёный |
| D | Прогноз_f (шт) | Ввод | Прогнозные продажи | 🟢 Зелёный |
| E | Ошибка_e | Формула | =C-D | ⚪ Серый |
| F | \|e\| | Формула | =ABS(E) | ⚪ Серый |
| G | %AbsErr | Формула | =IF(C>0;F/C;NA()) | ⚪ Серый |
| H | Примечание | Ввод | Комментарии | 🟢 Зелёный |

**Формулы (строка 2-1000):**
```excel
E2: =C2-D2                    // Ошибка прогноза
F2: =ABS(E2)                  // Абсолютная ошибка
G2: =IF(C2>0;F2/C2;NA())      // % ошибка (защита от деления на 0)
```

**Условное форматирование:**
- 🟡 Жёлтая заливка → C=0 (нулевой факт, MAPE не считается)
- 🟠 Оранжевая заливка → F в топ-10% (выбросы по ошибке)

**Валидация данных:**
- C, D ≥ 0 (только положительные числа)
- Пустые ячейки разрешены
- Автоматическое предупреждение при ошибках

**Именованные диапазоны:**
- `rngY` → $C$2:$C$1000 (факт)
- `rngF` → $D$2:$D$1000 (прогноз)
- `rngE` → $E$2:$E$1000 (ошибка)
- `rngAbsE` → $F$2:$F$1000 (абсолютная ошибка)
- `rngPctAbsE` → $G$2:$G$1000 (% ошибка)
- `rngSKU` → $A$2:$A$1000 (SKU для фильтров)

**Тестовые данные:**
- 12 периодов (месяцы 2024 года)
- 1 SKU (SKU_TEST_001)
- Реалистичные значения с MAPE ~8-12%

### 3. 📈 МЕТРИКИ
**Назначение:** Автоматический расчёт метрик качества прогноза

**Таблица метрик:**

| Метрика | Формула Excel | Единицы | Интерпретация |
|---------|---------------|---------|---------------|
| **MAPE** | `=IFERROR(AVERAGE(IF(ISNUMBER(rng);rng));NA())` | % | Средняя относительная ошибка |
| **WAPE** | `=IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())` | % | Взвешенная ошибка по объёму |
| **MAE** | `=IFERROR(AVERAGE(IF(rngAbsE<>"";rngAbsE));NA())` | шт | Средняя абсолютная ошибка |
| **RMSE** | `=IFERROR(SQRT(AVERAGE((rngE)^2));NA())` | шт | Усиливает влияние больших промахов |
| **Bias** | `=IFERROR(AVERAGE(IF(rngE<>"";rngE));NA())` | шт | Знак = направление смещения |

**Диагностика:**

1. **Знак Bias:**
   ```excel
   =IF(B7>0;"Недопрогноз (дефициты)";IF(B7<0;"Перепрогноз (излишки)";"Смещения нет"))
   ```

2. **Сравнение RMSE vs MAE:**
   ```excel
   =IF(B6>B5;"Есть выбросы/крупные промахи";"Ошибки равномерны")
   ```

**Рекомендации (формульные):**
- Высокий WAPE → "Проверьте сезонность/акции/качество данных"
- RMSE >> MAE → "Сгладьте выбросы или разнесите всплески"
- |Bias| значим → "Сместите модель/перепараметризуйте"

**Условное форматирование для WAPE:**
- 🟢 Зелёный: < 10% (отлично)
- 🟡 Янтарный: 10-15% (средне)
- 🔴 Красный: > 15% (требует улучшения)

### 4. 📊 ГРАФИКИ
**Назначение:** Визуальный анализ качества прогноза

**График 1: Факт vs Прогноз**
- Тип: Линейный график
- Оси: Период (X), Объём в штуках (Y)
- Данные: Первые 50 строк из листа Ввод_данных
- Легенда: Факт_y (синяя линия), Прогноз_f (оранжевая линия)

**График 2: Абсолютная ошибка по периодам**
- Тип: Столбчатая диаграмма
- Оси: Период (X), Абсолютная ошибка (Y)
- Данные: Колонка F (|e|) из листа Ввод_данных
- Позволяет визуально выявить выбросы

**Карта статусов (панель справа):**
- WAPE: текущее значение с %
- Bias: текущее значение
- Статус: 
  - ✓ Отлично (зелёный) — WAPE < 10%
  - ⚠ Средне (янтарный) — WAPE 10-15%
  - ✗ Требует улучшения (красный) — WAPE > 15%

**Интерактивность:**
- Автообновление при изменении данных
- Условное форматирование для статуса
