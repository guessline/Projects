# Погодный агент в Telegram ☀️

Каждый день в **06:30 по московскому времени** присылает в Telegram сводку
погоды по двум городам — **Москве** и **Стерлитамаку**: текущая температура,
ощущаемая температура, влажность, ветер, прогноз на день (мин/макс, вероятность
осадков), время восхода и заката.

Данные берутся с [Open-Meteo](https://open-meteo.com/) — бесплатно и без
API-ключа. Запуск по расписанию выполняет GitHub Actions
(`.github/workflows/weather-telegram.yml`).

## Как настроить (один раз)

### 1. Создайте Telegram-бота

1. Откройте в Telegram чат с [@BotFather](https://t.me/BotFather).
2. Отправьте `/newbot` и следуйте инструкциям (имя и username бота).
3. BotFather пришлёт **токен** вида `123456789:AAE...` — сохраните его.

### 2. Узнайте свой chat_id

1. Напишите своему новому боту любое сообщение (например, `/start`) — без
   этого бот не сможет вам писать.
2. Откройте в браузере (подставив токен):
   `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
3. Найдите в ответе `"chat":{"id":...}` — это ваш **chat_id**.

### 3. Добавьте секреты в репозиторий GitHub

В репозитории: **Settings → Secrets and variables → Actions → New repository
secret**. Создайте два секрета:

| Имя секрета          | Значение                          |
| -------------------- | --------------------------------- |
| `TELEGRAM_BOT_TOKEN` | токен от BotFather                |
| `TELEGRAM_CHAT_ID`   | ваш chat_id                       |

### 4. Проверьте работу

Откройте вкладку **Actions → «Утренняя погода в Telegram» → Run workflow**,
чтобы запустить отправку вручную прямо сейчас. Если всё настроено верно — придёт
сообщение в Telegram. Дальше агент будет срабатывать сам каждый день в 06:30 МСК.

> ⚠️ GitHub Actions запускает задачи по расписанию в UTC и в периоды высокой
> нагрузки может задержать запуск на несколько минут — это нормально.

## Запуск вручную (локально)

```bash
pip install -r weather_telegram_agent/requirements.txt
export TELEGRAM_BOT_TOKEN="123456789:AAE..."
export TELEGRAM_CHAT_ID="123456789"
python weather_telegram_agent/weather_bot.py
```

## Как изменить города

Отредактируйте список `CITIES` в файле `weather_telegram_agent/weather_bot.py`,
указав название и координаты (широту/долготу) нужного города.
