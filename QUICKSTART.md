# 🚀 БЫСТРЫЙ СТАРТ — Forecast Metrics Template

## 📊 Что это?

Excel-шаблон для оценки качества прогнозирования через 5 ключевых метрик:
- **MAPE** — средняя относительная ошибка (%)
- **WAPE** — взвешенная ошибка по объёму (%)
- **MAE** — средняя абсолютная ошибка (штуки)
- **RMSE** — среднеквадратичная ошибка (штуки)
- **Bias** — систематическое смещение (штуки)

---

## ⚡ 3 минуты до результата

### Шаг 1: Откройте файл
```
Forecast_Metrics_Template.xlsx
```

### Шаг 2: Вставьте данные
Перейдите на лист **"Ввод_данных"** и заполните:
- **Колонка C** (Факт_y) — фактические продажи
- **Колонка D** (Прогноз_f) — прогнозные продажи

### Шаг 3: Смотрите результаты
Перейдите на лист **"Метрики"** — все показатели рассчитаны автоматически!

### Шаг 4: Анализируйте графики
Лист **"Графики"** покажет визуальное сравнение и выбросы.

## 🎯 Как читать результаты

### WAPE (главная метрика)
- 🟢 **< 10%** — отличный прогноз
- 🟡 **10-15%** — средний прогноз
- 🔴 **> 15%** — нужно улучшить

### Bias (направление ошибки)
- **> 0** → Недопрогноз → риск дефицитов
- **< 0** → Перепрогноз → риск излишков
- **≈ 0** → Сбалансированно (но проверьте WAPE!)

### RMSE vs MAE
- **RMSE ≈ MAE** → ошибки равномерны
- **RMSE >> MAE** → есть выбросы, проверьте оранжевые ячейки

## 🎨 Цвета (что они значат)

| Цвет | Где | Значение |
|------|-----|----------|
| 🟢 Светло-зелёный | Ввод_данных | Можно редактировать |
| ⚪ Серый | Ввод_данных | Формулы (защищены) |
| 🟡 Жёлтый | Ввод_данных | y=0, MAPE не считается |
| 🟠 Оранжевый | Ввод_данных | Выброс (топ-10% ошибок) |
| 🟢/🟡/🔴 | Метрики | Качество WAPE |

---

## ⚠️ Важно знать

### MAPE не работает при нулях
Если в данных много нулевых продаж (>30%), смотрите **WAPE** вместо MAPE.

### Малый Bias ≠ хороший прогноз
Ошибки могут компенсироваться! Всегда проверяйте WAPE или MAE.

### Выбросы
Оранжевые ячейки — это топ-10% ошибок. Проверьте их причины.
