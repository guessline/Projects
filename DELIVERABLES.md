## ✅ Основные файлы

### 1. 📊 `Forecast_Metrics_Template.xlsx` (60 KB)
**Основной Excel-шаблон для расчёта метрик прогнозирования**

#### Структура:
- **Лист 1: Инструкции** — Подробное руководство пользователя, описание метрик, ограничения
- **Лист 2: Ввод_данных** — Таблица ввода с автоформулами, валидацией и условным форматированием

- **Лист 4: Графики** — Визуализация: Факт vs Прогноз, ошибки, карта статусов

#### Возможности:
✅ Автоматический расчёт 5 метрик  
✅ Защита от ошибок (#DIV/0!, пустые ячейки)  
✅ Условное форматирование (нули, выбросы, статусы)  
✅ Защита листов с расчётами  
✅ Валидация ввода (только числа ≥ 0)  
✅ Тестовые данные (12 периодов)  
✅ Именованные диапазоны для удобства  

#### Именованные диапазоны:
- `rngY` — Факт_y (C2:C1000)
- `rngF` — Прогноз_f (D2:D1000)
- `rngE` — Ошибка_e (E2:E1000)
- `rngAbsE` — |e| (F2:F1000)
- `rngPctAbsE` — %AbsErr (G2:G1000)
- `rngSKU` — SKU (A2:A1000)

## 📊 ФОРМУЛЫ EXCEL (Quick Reference)

### Лист "Ввод_данных"

```excel
; Ошибка (E2)
=C2-D2

; Абсолютная ошибка (F2)
=ABS(E2)

; Процентная ошибка (G2)
=IF(C2>0;F2/C2;NA())
```

### Лист "Метрики"

```excel
; MAPE — средняя процентная ошибка
=IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA())

; WAPE — взвешенная процентная ошибка
=IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())

; MAE — средняя абсолютная ошибка
=IFERROR(AVERAGE(IF(rngAbsE<>"";rngAbsE));NA())

; RMSE — среднеквадратичная ошибка
=IFERROR(SQRT(AVERAGE((rngE)^2));NA())

; Bias — смещение
=IFERROR(AVERAGE(IF(rngE<>"";rngE));NA())

; Интерпретация Bias
=IF(B8>0;"Недопрогноз (дефициты)";IF(B8<0;"Перепрогноз (излишки)";"Смещения нет"))

; RMSE vs MAE
=IF(B7>B6;"Есть выбросы/крупные промахи";"Ошибки равномерны")

; Рекомендация: Высокий WAPE
=IF(B5>0.15;"⚠ Проверьте сезонность/акции/качество данных";"-")

; Рекомендация: Выбросы
=IF(B7>B6*1.2;"⚠ Сгладьте выбросы или разнесите всплески";"-")

; Рекомендация: Значимый Bias
=IF(ABS(B8)>B6*0.3;"⚠ Сместите модель/перепараметризуйте";"-")
```
