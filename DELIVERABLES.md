## 📊 Реализованные функции

### ✅ Формулы и расчеты
- [x] **MAPE** - средняя абсолютная процентная ошибка с защитой от нулей
- [x] **WAPE** - взвешенная абсолютная процентная ошибка
- [x] **MAE** - средняя абсолютная ошибка
- [x] **RMSE** - среднеквадратичная ошибка
- [x] **Bias** - систематическое смещение прогноза

### ✅ Автоматизация
- [x] Именованные диапазоны (rngY, rngF, rngE, rngAbsE, rngPctAbsE, rngSKU)
- [x] Автоматические формулы на 1000 строк
- [x] Условное форматирование:
  - Жёлтая подсветка нулей в факте
  - Оранжевая подсветка выбросов (топ-10%)
  - Цветовая индикация метрик (зелёный/янтарный/красный)
  - Синий/фиолетовый для Bias

### ✅ Валидация и защита
- [x] Валидация данных: C:D ≥ 0 (только положительные числа)
- [x] Защита всех листов (без пароля)
- [x] Редактируемые поля: A, B, C, D, H на листе "Ввод_данных"
- [x] Защита формул от изменения

### ✅ UX и оформление
- [x] Цветовая схема:
  - Светло-зелёный (#E2F0D9) - поля ввода
  - Серый (#F2F2F2) - вычисляемые поля
  - Тёмно-серый (#404040) - заголовки
- [x] Шрифт Calibri 11
- [x] Заморозка верхней строки
- [x] Автоширина колонок
- [x] Комментарии-подсказки в заголовках

### ✅ Диагностика и интерпретация
- [x] Автоматическое определение знака Bias
- [x] Сравнение RMSE vs MAE для выявления выбросов
- [x] Рекомендации по действиям на основе метрик
- [x] Легенда цветов и инструкции на отдельном листе

### ✅ Графики
- [x] Линейный график "Факт vs Прогноз"
- [x] Столбчатая диаграмма "Абсолютная ошибка по периодам"
- [x] Автоматическое обновление при добавлении данных

### ✅ Тестовые данные
- [x] 12 периодов тестовых данных (SKU-001)

  - WAPE: 4.33% (отлично!)

  - RMSE: 5.7 шт

## 🚀 Как использовать

1. **Откройте** `Forecast_Metrics_Template.xlsx` в Excel или Google Sheets
2. **Прочитайте** лист "Инструкции"
3. **Замените** тестовые данные на свои на листе "Ввод_данных"
4. **Анализируйте** результаты на листе "Метрики"
5. **Визуализируйте** данные на листе "Графики"
