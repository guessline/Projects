## 🚀 Быстрый старт

### Использование готового шаблона:

1. Откройте **Forecast_Metrics_Template.xlsx**
2. Перейдите на лист "Ввод_данных"
3. Заполните колонки C (Факт) и D (Прогноз)
4. Просмотрите результаты на листе "Метрики"
5. Изучите графики на листе "Графики"

### Пересоздание шаблона:

```bash
# Установка зависимостей
pip install openpyxl

# Генерация файла
python3 generate_forecast_template.py

# Тестирование
python3 test_qa_checklist.py
```

## 📊 Ключевые возможности

✅ **Автоматический расчёт метрик:**
- MAPE - средняя процентная ошибка
- WAPE - взвешенная процентная ошибка  
- MAE - средняя абсолютная ошибка
- RMSE - среднеквадратичная ошибка
- Bias - систематическое смещение

✅ **Защита от ошибок:**
- Формулы защищены от деления на ноль
- Валидация ввода (только положительные числа)
- IFERROR обработка пустых данных

✅ **Визуальные индикаторы:**
- Жёлтая подсветка нулевых значений
- Оранжевая подсветка выбросов (топ-10%)
- Цветовая шкала для WAPE (зелёный/янтарный/красный)
- Индикатор Bias (синий/фиолетовый)

✅ **Диагностика и рекомендации:**
- Автоматическое определение недопрогноза/перепрогноза
- Выявление выбросов через сравнение RMSE vs MAE
- Рекомендации по улучшению прогноза

✅ **Графики:**
- Факт vs Прогноз (линейный график)
- Абсолютная ошибка по периодам (столбчатый)
- Карта статусов метрик
