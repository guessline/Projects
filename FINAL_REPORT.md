## 🎓 Использование

### Для кого этот шаблон

| Роль | Применение |
|------|------------|
| **Аналитик данных** | Оценка точности моделей |
| **Supply Chain** | Контроль качества прогнозов |
| **Менеджер продаж** | Анализ отклонений плана/факта |
| **Руководитель** | Отчётность по KPI прогноза |
| **Студент/исследователь** | Изучение метрик прогнозирования |

### Метрики приоритета

1. **WAPE** — главная метрика для бизнеса и отчётности
2. **Bias** — всегда проверяйте направление ошибки
3. **MAE** — простая и понятная оценка
4. **RMSE vs MAE** — поиск выбросов
5. **MAPE** — только для стабильных рядов без нулей
