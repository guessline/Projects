## 🔑 Ключевые особенности

### 1. Формулы с защитой от ошибок
```excel
G2: =IF(C2>0;F2/C2;NA())              // Защита от деления на 0
MAPE: =IFERROR(AVERAGE(...);NA())     // Обработка пустых данных
WAPE: =IF(SUM(y)>0;...;NA())          // Проверка суммы
```

### 2. Условное форматирование
- **Ввод_данных**: жёлтая (y=0), оранжевая (выбросы)
- **Метрики**: зелёная/янтарная/красная (WAPE <10%/10-15%/>15%)
- **Графики**: зелёная/красная (статус)

### 3. Именованные диапазоны
```
rngY → Ввод_данных!$C$2:$C$1000      (факт)
rngF → Ввод_данных!$D$2:$D$1000      (прогноз)
rngE → Ввод_данных!$E$2:$E$1000      (ошибка)
rngAbsE → Ввод_данных!$F$2:$F$1000   (|e|)
rngPctAbsE → Ввод_данных!$G$2:$G$1000 (%AbsErr)
rngSKU → Ввод_данных!$A$2:$A$1000    (SKU)
```

### 4. Диагностика
- **Знак Bias**: Недопрогноз/Перепрогноз/Нет смещения
- **RMSE vs MAE**: Выбросы/Равномерные ошибки
- **Рекомендации**: Что делать при высоких значениях

### 5. Визуализация
- График сравнения факта и прогноза
- Диаграмма ошибок по периодам
- Карта статусов с автооценкой качества

## 🎓 Интерпретация метрик

### MAPE (Mean Absolute Percentage Error)
- **Что показывает**: Средняя относительная ошибка в %
- **Когда использовать**: Стабильные ряды без нулей
- **Норма**: <10% — отлично, 10-20% — хорошо, >20% — плохо
- **⚠️ Ограничение**: Не работает при y=0

### WAPE (Weighted Absolute Percentage Error)
- **Что показывает**: Взвешенная ошибка по объёму
- **Когда использовать**: Портфели, отчёты руководству
- **Норма**: <10% — зелёный, 10-15% — янтарный, >15% — красный
- **✅ Преимущество**: Устойчива к выбросам и нулям

### MAE (Mean Absolute Error)
- **Что показывает**: Типичная ошибка в натуральных единицах
- **Когда использовать**: Для понимания величины в штуках
- **Интерпретация**: "В среднем ошибаемся на X штук"

### RMSE (Root Mean Squared Error)
- **Что показывает**: Усиленное влияние больших ошибок
- **Когда использовать**: Когда большие ошибки критичнее малых
- **Интерпретация**: RMSE >> MAE → есть выбросы

### Bias (Систематическое смещение)
- **Что показывает**: Направление ошибки (знак важен!)
- **Когда использовать**: ВСЕГДА проверяйте!
- **Интерпретация**:
  - **> 0**: Недопрогноз → дефициты
  - **< 0**: Перепрогноз → излишки
  - **≈ 0**: Ошибки компенсируются (но могут быть большими!)

---

## ⚠️ Важные ограничения

### 1. MAPE при нулях
```
Если доля нулей > 30% → используйте WAPE вместо MAPE
```

### 2. Выбросы
```
RMSE сильно реагирует на выбросы → проверьте оранжевые ячейки
```

### 3. Bias ≈ 0
```
Малый |Bias| НЕ означает хороший прогноз!
Ошибки могут компенсироваться.
Всегда смотрите MAPE/WAPE вместе с Bias.
```
