## 🚀 Что это за проект?

**Forecast Metrics Template** — это профессиональный Excel-шаблон для автоматического расчёта и анализа точности прогнозов.

### Основные возможности:
- ✅ Расчёт 5 ключевых метрик (MAPE, WAPE, MAE, RMSE, Bias)
- ✅ Автоматическая диагностика и рекомендации
- ✅ Визуализация данных (графики)
- ✅ Защита формул от случайных изменений
- ✅ Условное форматирование (подсветка ошибок)
- ✅ Встроенные инструкции и подсказки

### Кому полезен:
- 📊 Аналитикам данных
- 📈 Специалистам по прогнозированию
- 🏭 Supply chain менеджерам
- 💼 Руководителям отделов продаж
- 🎓 Студентам и исследователям

## 📊 Ключевые метрики - Краткая справка

| Метрика | Формула | Когда использовать |
|---------|---------|-------------------|
| **MAPE** | `AVG(\|e\|/y)` | Стабильные ряды без нулей |
| **WAPE** | `SUM(\|e\|)/SUM(y)` | Портфели, отчётность, есть нули |
| **MAE** | `AVG(\|e\|)` | Простая оценка в штуках |
| **RMSE** | `SQRT(AVG(e²))` | Поиск выбросов |
| **Bias** | `AVG(e)` | Проверка смещения (+/-) |

**Приоритет для большинства задач:** WAPE → Bias → MAE

## 🎨 Цветовая легенда

| Цвет | Где | Значение |
|------|-----|----------|
| 🟩 Светло-зелёный | Ввод | Ячейки для заполнения |
| ⬜ Серый | Расчёт | Формулы (не трогать) |
| 🟨 Жёлтый | Предупреждение | y=0, MAPE не считается |
| 🟧 Оранжевый | Выбросы | Топ-10% ошибок |
| 🟩 Зелёный | Метрики | WAPE <10% (хорошо) |
| 🟨 Янтарный | Метрики | WAPE 10-15% (приемлемо) |
| 🟥 Красный | Метрики | WAPE >15% (плохо) |
| 🔵 Синий | Bias | Недопрогноз (дефициты) |
| 🟣 Фиолетовый | Bias | Перепрогноз (излишки) |

## ❓ FAQ - Частые вопросы

### Q: С чего начать?
**A:** Откройте [README.md](README.md), затем Excel файл.

### Q: Какая метрика самая важная?
**A:** **WAPE** для большинства бизнес-задач, **MAPE** для сравнения моделей.

### Q: Что делать, если WAPE >15%?
**A:** См. [QUICK_GUIDE.md](QUICK_GUIDE.md) → "Быстрые исправления" → "Высокий WAPE"

### Q: Почему MAPE показывает #Н/Д?
**A:** Есть нулевые фактические значения. Используйте WAPE.

### Q: Как снять защиту с листов?
**A:** Пароль: `forecast2024` (или пересоздайте без защиты через Python)

### Q: Работает ли в Google Sheets?
**A:** Да, но нужно заменить `;` на `,` в формулах.

### Q: Можно ли использовать для нескольких SKU?
**A:** Да, заполняйте колонку SKU. Для агрегации по SKU см. версию 2.0 (планируется).

### Q: Как пересоздать шаблон?
**A:** `pip install openpyxl && python3 create_forecast_template.py`
