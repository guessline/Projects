## 📊 Метрики

Шаблон рассчитывает 5 ключевых метрик:

### 1. **MAPE** (Mean Absolute Percentage Error)
- **Формула**: `AVERAGE(|y-f|/y) × 100%`
- **Интерпретация**: Средняя относительная ошибка в процентах
- **Когда применять**: Для стабильных рядов без нулей
- **Целевой уровень**: < 10% — отлично, 10-20% — хорошо

### 2. **WAPE** (Weighted Absolute Percentage Error)
- **Формула**: `SUM(|y-f|) / SUM(y) × 100%`
- **Интерпретация**: Взвешенная ошибка с учётом объёма
- **Когда применять**: Для портфелей товаров, отчётности
- **Целевой уровень**: < 10% 🟢 | 10-15% 🟠 | > 15% 🔴

### 3. **MAE** (Mean Absolute Error)
- **Формула**: `AVERAGE(|y-f|)`
- **Интерпретация**: Средняя абсолютная ошибка в штуках
- **Когда применять**: Для операционного планирования

### 4. **RMSE** (Root Mean Square Error)
- **Формула**: `SQRT(AVERAGE((y-f)²))`
- **Интерпретация**: Усиливает влияние больших ошибок
- **Когда применять**: Для выявления выбросов
- **Правило**: RMSE >> MAE → есть выбросы

### 5. **Bias** (Смещение)
- **Формула**: `AVERAGE(y-f)`
- **Интерпретация**: Направление систематической ошибки
- **Знак**:
  - Bias > 0 🔵 → недопрогноз (дефициты)
  - Bias < 0 🟣 → перепрогноз (излишки)
  - Bias ≈ 0 → смещения нет

## ⚠️ Важные замечания

### MAPE при нулевых значениях
> **Правило**: Если доля нулей в факте > 30%, не используйте MAPE  
> **Альтернатива**: Используйте WAPE или MAE

### Влияние выбросов на RMSE
> RMSE более чувствителен к выбросам, чем MAE  
> **Индикатор**: Если RMSE >> MAE → есть крупные промахи

### Смысл Bias
> Низкий Bias ≠ хороший прогноз  
> Ошибки могут компенсировать друг друга  
> **Правило**: Анализируйте Bias вместе с WAPE/MAE/RMSE
