### Лист 2: "Ввод_данных"

**Колонки:**

| Колонка | Название | Тип | Описание |
|---------|----------|-----|----------|
| A | SKU | Ввод | Артикул/код товара |
| B | Период | Ввод | Дата или номер периода |
| C | Факт_y | Ввод | Фактические продажи (шт) |
| D | Прогноз_f | Ввод | Прогнозные значения (шт) |
| E | Ошибка_e | Формула | =C-D (разница) |
| F | Абс_Ошибка | Формула | =ABS(E) |
| G | Проц_Ошибка | Формула | =IF(C>0;F/C;NA()) |
| H | Примечание | Ввод | Доп. информация |

**Формулы (строки 2-1000):**
```excel
E2: =C2-D2
F2: =ABS(E2)
G2: =IF(C2>0;F2/C2;NA())
```

**Форматирование:**
- Заголовки: тёмно-серый фон (#404040), белый шрифт
- Поля ввода (A, B, C, D, H): светло-зелёный фон (#E2F0D9)
- Вычисляемые поля (E, F, G): серый фон (#F2F2F2)
- Верхняя строка заморожена

**Валидация данных:**
- C2:C1000 и D2:D1000: значения ≥ 0 (decimal)
- Пустые значения разрешены

**Условное форматирование:**
1. C2:C1000 - жёлтая заливка при C=0 (предупреждение о нулях)
2. F2:F1000 - оранжевая заливка для топ-10% ошибок (выбросы)

**Примечания-подсказки:**
- Каждый заголовок содержит комментарий с описанием

**Тестовые данные:**
- 12 периодов для SKU-001
- Факт: от 98 до 140 шт
- Прогноз: от 92 до 140 шт
- WAPE ≈ 3.81%

### Лист 3: "Метрики"

**Таблица метрик:**

| Метрика | Формула | Единицы | Интерпретация | Примечание |
|---------|---------|---------|---------------|------------|
| MAPE | =IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA()) | % | Средняя относительная ошибка | Не рассчитывается при y=0 |
| WAPE | =IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA()) | % | Взвешенная ошибка по объёму | Устойчива к выбросам |
| MAE | =IFERROR(AVERAGEIF(rngAbsE;">0");NA()) | шт | Средняя абсолютная ошибка | Простая интерпретация |
| RMSE | =IFERROR(SQRT(AVERAGE(rngE^2));NA()) | шт | Среднеквадратичная ошибка | Усиливает влияние промахов |
| Bias | =IFERROR(AVERAGE(rngE);NA()) | шт | Систематическое смещение | Знак = направление |

**Диагностика:**
- Знак Bias: определяет недопрогноз/перепрогноз
- Сравнение RMSE vs MAE: выявляет выбросы

**Рекомендации (автоматические):**
- При высоком WAPE (>15%)
- При RMSE >> MAE (коэф. >1.5)
- При значимом |Bias| (>30% от MAE)

**Условное форматирование:**
- WAPE (B5):
  - Зелёный: < 10%
  - Янтарный: 10-15%
  - Красный: > 15%
- Bias (B8):
  - Синий: > 0 (недопрогноз)
  - Фиолетовый: < 0 (перепрогноз)

**Именованные ячейки:**
- MAPE_value → Метрики!B4
- WAPE_value → Метрики!B5
- MAE_value → Метрики!B6
- RMSE_value → Метрики!B7
- Bias_value → Метрики!B8

## 📝 Примечания к реализации

### Особенности формул:

1. **MAPE с массивом:**
   ```excel
   =IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA())
   ```
   - Массивная формула (Ctrl+Shift+Enter в старых версиях Excel)
   - Пропускает #Н/Д значения при y=0

2. **WAPE с защитой:**
   ```excel
   =IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())
   ```
   - Проверка деления на ноль
   - Возвращает NA() если нет данных

3. **RMSE через квадраты:**
   ```excel
   =IFERROR(SQRT(AVERAGE(rngE^2));NA())
   ```
   - Работает с именованным диапазоном
   - IFERROR обрабатывает пустые данные
