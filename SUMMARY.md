# ✅ ИТОГОВАЯ СВОДКА ПО ПРОЕКТУ

## Задача
Создание Excel-шаблона `Forecast_Metrics_Template.xlsx` для расчёта и интерпретации метрик прогнозирования (MAPE, WAPE, MAE, RMSE, Bias).

## Выполненные задачи

### ✅ 1. Создан Excel-шаблон с 4 листами:

#### 📋 Лист "Инструкции"
- Подробное руководство пользователя
- Описание всех метрик с формулами
- Правила интерпретации результатов
- Ограничения и важные замечания
- Легенда цветов
- Навигация по листам

#### 📊 Лист "Ввод_данных"
- Таблица с 8 колонками (A-H)
- Защита от некорректного ввода (валидация ≥ 0)
- Автоматические формулы:
  - `E: Ошибка_e = C - D`
  - `F: |e| = ABS(E)`
  - `G: %AbsErr = IF(C>0; F/C; NA())`
- Условное форматирование:
  - 🟡 Жёлтая заливка при C=0 (MAPE недоступен)
  - 🟠 Оранжевая заливка для выбросов (топ-10%)
- Тестовые данные: 12 периодов по 1 SKU
- Защита: разрешено редактирование только C, D, H

#### 📈 Лист "Метрики"
- Расчёт 5 метрик с формулами:
  - **MAPE**: `=IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA())`
  - **WAPE**: `=IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())`
  - **MAE**: `=IFERROR(AVERAGE(IF(rngAbsE<>"";rngAbsE));NA())`
  - **RMSE**: `=IFERROR(SQRT(AVERAGE((rngE)^2));NA())`
  - **Bias**: `=IFERROR(AVERAGE(IF(rngE<>"";rngE));NA())`
- Диагностика:
  - Интерпретация знака Bias
  - Сравнение RMSE vs MAE
  - Автоматические рекомендации
- Условное форматирование:
  - 🟢 WAPE < 10% (отлично)
  - 🟠 WAPE 10-15% (приемлемо)
  - 🔴 WAPE > 15% (требует улучшения)
  - 🔵 Bias > 0 (недопрогноз)
  - 🟣 Bias < 0 (перепрогноз)
- Полная защита листа

#### 📉 Лист "Графики"
- График 1: Линейный "Факт vs Прогноз"
- График 2: Столбчатый "Абсолютная ошибка по периодам"
- Карта статусов метрик с индикаторами
- Полная защита листа

### ✅ 2. Именованные диапазоны
Созданы 6 именованных диапазонов для формул:
- `rngY` → Ввод_данных!C2:C1000 (Факт)
- `rngF` → Ввод_данных!D2:D1000 (Прогноз)
- `rngE` → Ввод_данных!E2:E1000 (Ошибка)
- `rngAbsE` → Ввод_данных!F2:F1000 (Абс. ошибка)
- `rngPctAbsE` → Ввод_данных!G2:G1000 (% ошибка)
- `rngSKU` → Ввод_данных!A2:A1000 (SKU)

## Примеры использования

### Сценарий 1: Анализ одного SKU
1. Вставьте данные в лист "Ввод_данных"
2. Проверьте метрики на листе "Метрики"
3. Изучите графики и рекомендации

### Сценарий 2: Портфельный анализ
1. Добавьте данные по нескольким SKU
2. Метрики рассчитаются по всему портфелю
3. WAPE покажет взвешенную ошибку с учётом объёмов

### Сценарий 3: Диагностика проблем
1. Найдите оранжевые ячейки (выбросы)
2. Проверьте Bias (направление смещения)
3. Сравните RMSE и MAE (равномерность ошибок)
4. Следуйте автоматическим рекомендациям

## Ограничения и рекомендации

⚠️ **MAPE при y=0**: Не рассчитывается. При доле нулей >30% используйте WAPE/MAE  
⚠️ **RMSE vs MAE**: Если RMSE >> MAE — есть выбросы, требуется сглаживание  
⚠️ **Bias**: Низкий Bias ≠ хороший прогноз. Анализируйте вместе с WAPE/MAE
