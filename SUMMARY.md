## 📊 Примеры формул

### Основные расчеты
```excel
Ошибка (E2):     =C2-D2
|Ошибка| (F2):   =ABS(E2)
%AbsErr (G2):    =IF(C2>0;F2/C2;NA())
```

### Метрики
```excel
MAPE (B4):  =IFERROR(AVERAGE(IF(ISNUMBER(rngPctAbsE);rngPctAbsE));NA())
WAPE (B5):  =IF(SUM(rngY)>0;SUM(rngAbsE)/SUM(rngY);NA())
MAE (B6):   =IFERROR(AVERAGEIF(rngAbsE;"<>0");NA())
RMSE (B7):  =IFERROR(SQRT(SUMPRODUCT(rngE;rngE)/COUNTIF(rngE;"<>0"));NA())
Bias (B8):  =IFERROR(AVERAGEIF(rngE;"<>0");NA())
```

## 📖 Интерпретация метрик

### WAPE (основная метрика)
- **< 10%** 🟢 - отличная точность
- **10-15%** 🟡 - приемлемая точность  
- **> 15%** 🔴 - требуется улучшение

### MAPE
- Используйте только если нулей < 30%
- **< 10%** - хорошо
- **10-20%** - приемлемо
- **> 20%** - плохо

### Bias (направление смещения)
- **> 0** 🔵 - недопрогноз (риск дефицитов)
- **< 0** 🟣 - перепрогноз (риск излишков)
- **≈ 0** - нет систематического смещения

### RMSE vs MAE
- **RMSE >> MAE** - есть выбросы, крупные промахи
- **RMSE ≈ MAE** - ошибки равномерны

---

## ⚠️ Важные ограничения

1. **MAPE** не рассчитывается для периодов с y=0
2. Если доля нулей **> 30%** - используйте **WAPE** вместо MAPE
3. **RMSE** чувствителен к выбросам - сравнивайте с MAE
4. Нужно минимум **5-10 периодов** для корректных выводов
